# tp5admin
tp5admin是一款基于thinkphp5.0框架开发的后台框架！
框架包含了基本的rbac权限！
包含了基本的网站配置功能，后台菜单管理功能，清除缓存和文章模块！
本框架不包含其他多余的功能，拿来做自己的后台基础，再适合不过了！
框架可以拿去做任何修改
